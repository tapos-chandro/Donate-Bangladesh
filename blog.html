What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree of objects, allowing developers to interact with and manipulate the content, structure, and styling of a website dynamically using programming languages like JavaScript.

How do you select an element from the DOM?

To select an element from the DOM:
Use getElementById to select an element by its unique ID.
Use getElementsByClassName to select elements with a specific class.
Use getElementsByTagName to select elements by their tag name.
Use querySelector for the first element matching a CSS selector.
Use querySelectorAll for all elements matching a CSS selector.
Use getElementsByName to select elements by their name attribute.
Each method provides a way to target and manipulate specific elements in the document.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a method where you attach an event listener to a parent element to manage events for its child elements. It works because events bubble up from the child to the parent, and the parent can identify the source using event.target.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using the DOM by accessing the element and using specific methods and properties.
